# Longest Palindrome

## requirement

Given a string s, find the longest palindromic substring in s

Example

1. longestPalindrome("cbbd") --> "bb"
2. longestPalindrome("abba") --> "abba"
3. longestPalindrome("a") --> "a"

## solution

### using mirror expand

1. time complexity: O($n^2$)
2. space complexity: O(1)
